# MF仕訳帳CSV形式サンプル

このドキュメントは、MoneyForward仕訳帳CSV取込で想定する最小フォーマットです。実際のMoneyForward出力に列が追加されていても、下記の必須列が含まれていれば取込対象にします。

## 文字コード

- UTF-8
- Shift_JIS

## 必須列

| 列名 | 用途 |
| --- | --- |
| 日付 | 仕訳日。`YYYY/MM/DD`、`YYYY-MM-DD`、`YYYY年M月D日` を許容 |
| 借方勘定科目 | 借方の勘定科目名 |
| 貸方勘定科目 | 貸方の勘定科目名 |
| 借方金額 / 貸方金額 / 金額 | 金額。カンマ区切り、円表記を許容 |

## 任意列

| 列名 | 用途 |
| --- | --- |
| 伝票番号 | 仕訳の伝票番号 |
| 借方補助科目 | 借方の補助科目名 |
| 貸方補助科目 | 貸方の補助科目名 |
| 借方部門 | 借方の部門名 |
| 貸方部門 | 貸方の部門名 |
| 部門 | 借方・貸方共通の部門名 |
| 借方税額 | 借方の税額 |
| 貸方税額 | 貸方の税額 |
| 摘要 | 摘要。カンマを含む場合はCSVの引用符で囲む |

## サンプル

```csv
日付,伝票番号,借方勘定科目,借方補助科目,借方部門,借方金額,借方税額,貸方勘定科目,貸方補助科目,貸方部門,貸方金額,貸方税額,摘要
2026/05/01,1001,普通預金,メイン口座,管理部,"110,000",0,売上高,商品売上,営業部,"110,000",10000,"4月売上,入金"
2026/05/02,1002,旅費交通費,,営業部,3300,300,現金,,営業部,3300,0,交通費
```

## 取込後の扱い

CSV取込は仕訳候補の登録までです。自動仕訳確定、税務判断の断定、AIによる自動確定は行いません。

取込後の明細は、仕訳確認画面で人間が確認します。CF分類結果が未作成の明細は、既存の画面上で未分類または要確認として扱われます。
